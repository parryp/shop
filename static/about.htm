About

This is a sample shop that integrates with PayPal, uses PayPal Express Check out. All transactions between this sample shop and to PayPal sandbox server. To try, create a PayPal sandbox account here.

A successful shopping flow to purchase is such,

  1. Browse and select products to add to cart.
  2. Check out, review list of items in shopping cart.
  3. PayPal Check out
    • SetExpressCheckout
      1. Sets up the Express Checkout transaction.
      2. Define a return url for PayPal to re-direct user's browser back to merchant shop
      3. Define a cancel url back to merchant shop; should buyer cancels.
      4. Retrieve a token, to identify with PayPal on further communication with this transaction
    • GetExpressCheckoutDetails
      • Use a token to make the request
      • PayPal respond with information about the buyer
    • Re-direct to PayPal with the token
      1. Login with a PayPal (sandbox) account
      2. Review item to purchase, contact details and payment method.
      3. Continue -> re-directs back to Achmed Shop of wonders to confirm the order and furnish merchant with a delivery address.
  4. Confirm Order
    • DoExpressCheckout
      1. Makes payment to merchant via PayPal express check out
  5. Order confirm

    6. This is a test shop that makes use of PayPal express checkout and is by no means a real shop.
